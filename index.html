Instapaper for Opera

Simple extension to add webpages to Instapaper.

This project is maintained by ramkumarshankar

My first attempt at a browser extension. A dead simple way to add webpages to Instapaper. For Opera.

How to use

On Opera, install it at https://addons.opera.com/en/addons/extensions/details/instapaper-for-opera/.

Go to the extension preferences and set it up with your Instapaper username and password (if you have one). When you're on a page, click the button to read it later.

You can also select some text, and it'll be added as a description along with the item.
